# MicroBankingSystem
마이크로 뱅킹 시스템

## 📋 프로젝트 설명
### 은행원을 위한 마이크로뱅킹 시스템
- 고객이 은행에 방문하여 요청하는 업무 처리 및 지점 마감 관리를 전산적으로 할 수 있도록 도와주는 시스템입니다.
  - 고객 및 행원 관리
  - 예적금의 가입부터 이자 관리 및 해지까지의 절차
  - 현금 입출금 및 계좌이체
  - OTP 인증
  - 지점의 현금 거래에 대한 마감 관리
